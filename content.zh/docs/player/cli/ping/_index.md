## Scaffold Ping

### NAME

main.py ping - 测试 RedisNode 连接

### SYNOPSIS

```bash
main.py ping - 
```

### DESCRIPTION

测试数据库连接。这是个不常用的功能，仅会在新环境首次创建时使用，如 在 GitHub Actions 中检查配置是否正确；在本地初次调试时，检查配置文件中 `REDIS_NODE` 是否配置正确等。
